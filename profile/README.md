## Overview

CodeRunner Advanced is a lightweight code editor and runner that enables writing and immediately executing code in over 25 programming languages with syntax highlighting, code completion, and integrated console output, making it the fastest path from idea to running code on macOS. It fills the gap between heavyweight IDEs that take minutes to configure a project and basic text editors that require terminal switching to run anything, providing an environment where code executes with a single keystroke. The application is ideal for scripting, prototyping, learning, and any coding task where immediate execution matters more than project-scale tooling.

Multi-language support runs Python, JavaScript, Ruby, Swift, C, C++, Go, Rust, PHP, Perl, shell scripts, and many more languages out of the box without requiring manual interpreter or compiler configuration. Each language runtime is detected automatically from the system, and custom run configurations can be defined for specialized toolchains or language versions. Language switching is instant — create a new file, select the language, write code, and run it in seconds.

The integrated console displays program output, accepts standard input, and shows error messages with clickable references to the source line that generated them, keeping the development feedback loop entirely within the application. Console history preserves output across multiple runs for comparison, and the console supports ANSI color output for programs that use terminal colors. Interactive programs that require user input work naturally through the integrated console.

Syntax highlighting with code completion and bracket matching provides a comfortable editing experience that covers the essentials of code editing without the configuration overhead of a full IDE. Custom themes control the editor appearance, and the completion engine draws from language-specific keyword lists and identifiers present in the current file. Document-wide search and replace with regex support handles the refactoring operations that quick coding tasks commonly require.

For developers, students, and scripters who want to write and run code immediately without project setup, build configuration, or IDE overhead, CodeRunner Advanced is the coding tool that prioritizes execution speed above all else. Its multi-language support and instant execution make it the go-to tool for quick scripts, algorithm prototyping, and language experimentation.

## Performance

CodeRunner balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
